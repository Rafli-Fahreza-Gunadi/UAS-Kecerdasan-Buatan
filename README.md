# Folder Referensi Jurnal

Folder ini berisi kumpulan jurnal ilmiah (PDF/link) yang digunakan sebagai referensi pada laporan (`Laporan_uas.md`), khususnya bagian Bab 9 - Referensi.

Silakan tambahkan file PDF jurnal Anda di sini, atau buat file `daftar_link_jurnal.txt` berisi tautan (link) ke masing-masing jurnal jika tidak ingin mengunggah file PDF secara langsung.
